# Images Folder

## 所需图片

### visual.png
- **用途**: Header组件的banner背景图
- **路径**: `src/assets/images/visual.png`
- **建议尺寸**: 1920x400px 或更大
- **格式**: PNG, JPG, 或 WebP
- **描述**: 会议主题banner图，包含"Urbanism & AI"主题元素和北京地标建筑

### 注意事项
1. 如果 visual.png 不存在，系统会自动使用渐变背景
2. 为获得最佳效果，请提供高分辨率图片
3. 图片应包含适当的主题元素和视觉层次
4. 建议图片宽度至少为 1920px 以支持大屏幕显示

## 添加图片的步骤

1. 将 `visual.png` 图片复制到此文件夹
2. 刷新浏览器，图片会自动加载
3. 如需更换图片，直接替换文件即可

## 其他图片

您可以在此文件夹中添加其他项目所需的图片：
- 演讲嘉宾照片
- 会议场地图片
- Logo 和图标
- 其他装饰性图片
